# Serein Wellness & Spa - Cameroon
A professional wellness platform infrastructure featuring automated security auditing and multi-currency support.

## 🚀 Key Features
* **Multi-Currency Support:** Real-time conversion between XAF, EUR, and USD.
* **Booking System:** Integrated service menu for Massages, Pedicures, and Wellness rituals.
* **DevSecOps Ready:** Automated Checkov security scans via GitHub Actions.
* **Local Integration:** Designed for Cameroon-specific payment workflows (MTN MoMo/Orange Money).

## 🛠 Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript.
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB (Mongoose).
* **Security:** Checkov, GitHub Actions.
